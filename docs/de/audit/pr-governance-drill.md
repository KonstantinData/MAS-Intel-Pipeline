# PR-Governance-Drill

Diese Datei dokumentiert einen kontrollierten Pull-Request-Drill zur Validierung von Branch-Schutz und erforderlichen CI-Gates.

## Umfang

- Validierung der erforderlichen Status-Checks für Pull Requests nach `main`
- Validierung der Durchsetzung von CODEOWNERS-Reviews
- Validierung der Merge-Blockierung, bis alle erforderlichen Checks erfolgreich sind
